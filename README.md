Mi Barrunto - Frontend package
-----------------------------
Este paquete contiene un frontend estático (HTML/CSS/JS) para la cevichería "Mi Barrunto".
Características:
- Diseño Alianza-inspired (azul/blanco), responsive.
- Menú cargado desde menu.json.
- Carrito de pedidos, simulación de caja (POS) y generación de comandas para cocina.
- Todo frontend (no hay backend). Para producción se necesita integrar APIs (pagos, orders, autenticación).

Datos rápidos utilizados para el diseño (extraídos de redes/listados públicos):
- Dirección: Jr. Sebastián Barranca 935, La Victoria.
- WhatsApp para pedidos (ejemplo): 942299769 / 984404540.
- Horario mostrado: Lun-Dom 10:00 - 17:00.
(Estas referencias se obtuvieron de redes sociales y listados públicos; ver fuentes proporcionadas en la conversación.)

Instrucciones:
- Descomprime el zip y abre index.html en un navegador.
- Para extender: conectar endpoints REST en app.js (fetch/save), o montar en Netlify/GitHub Pages.
